This is EXPERIMENT WEB,All HTML documents must start with a document type declaration: !DOCTYPE html .The HTML document itself begins with html and ends with .The visible part of the HTML document is between and .

The declaration represents the document type, and helps browsers to display web pages correctly.It must only appear once, at the top of the page (before any HTML tags).
